# v1.4 SSH 多人連入 — 實作版

## 目標

讓 k8s 中的每個 Docker 容器帳號都能被 SSH 連入，透過單一 IP + 不同埠號對應。

## 架構

```
Internet ── SSH ──► Host:50182 ──► Docker Container:22 (box5-ccc)
                 ──► Host:50xxx ──► Docker Container:22 (box5-ddd)
```

- 每個容器跑 SSH Server（埠 22）
- 主機動態分配埠號寫入 `uploads/{username}/.ssh_port`
- 容器內建立對應的系統使用者（`useradd`）
- 密碼與 FastAPI 密碼同步

## 變更檔案

| 檔案 | 變更 |
|------|------|
| `k8s/Dockerfile.box5` | 加入 openssh-server、sudo、start.sh、EXPOSE 22 |
| `k8s/start.sh` | 同時啟動 SSH daemon + FastAPI |
| `k8s/main.py` | `create_user_in_container()`, `get_ssh_port()`, expose SSH port, `/api/ssh/{username}` |
| `k8s/ccc_server.sh` | 一鍵啟動 + SSH 驗證腳本 |
| `k8s/test_ssh.sh` | 自動測試腳本 |

## 實作細節

### 1. Dockerfile.box5

- apt-get 安裝：`openssh-server sudo`
- SSH 設定：`/var/run/sshd` + `PasswordAuthentication yes`
- `start.sh` 複製進容器並設定執行權
- `EXPOSE 3111 22`

### 2. start.sh

```bash
#!/bin/bash
/usr/sbin/sshd
exec python -m uvicorn main:app --host 0.0.0.0 --port 3111
```

### 3. main.py — 關鍵函數

**SSH 埠常數**：`SSH_PORT=22`, `BASE_SSH_PORT=22000`

**create_user_in_container()**：在容器內建立系統使用者
1. `useradd -m -s /bin/zsh username`
2. `echo 'user:pass' | chpasswd`（若失敗則重試）
3. 驗證 `/etc/shadow` 確認密碼有設定（非 `!` 或空）

**get_ssh_port()**：讀取 `uploads/{username}/.ssh_port`，若無則 hash 計算

**容器 ports**：啟動時 expose `22/tcp`

**create_user_container()**：
1. 建立容器，ports 同時暴露 `3111/tcp` 和 `22/tcp`
2. 等 FastAPI 就緒
3. 等 SSH 就緒（socket connect test）
4. **建立 SSH 使用者（移到 SSH 就緒之後）**
5. 寫入 `.ssh_port` 檔案

### 4. /api/ssh/{username} API

```json
{"host": "localhost", "port": 50182, "username": "ccc", "command": "ssh ccc@localhost -p 50182"}
```

## 快速啟動

```bash
cd k8s && ./ccc_server.sh
```

輸出：
```
==================================================
Server:  http://localhost:8000
SSH:     ssh ccc@localhost -p 50182
Pass:    cccpass
==================================================
```

## 手動操作

```bash
# 重建 ccc 容器
docker rm -f box5-ccc
curl -X POST http://localhost:8000/login -d "username=ccc&password=cccpass"
sleep 25
cat uploads/ccc/.ssh_port

# SSH 連線
ssh ccc@localhost -p $(cat uploads/ccc/.ssh_port)
# 密碼：cccpass
```

## 安全考量

1. **防火牆**：只開放必要埠號（22 SSH、3111 API、8000 Web）
2. **密碼強度**：鼓勵使用者設定強密碼
3. **fail2ban**：防止暴力破解（future）

## 測試驗證

```bash
cd k8s && ./ccc_server.sh
# 輸出 SSH 連線指令後

# SSH 登入
ssh ccc@localhost -p $(cat uploads/ccc/.ssh_port)
# 密碼：cccpass

# SCP 上傳
scp -P $(cat uploads/ccc/.ssh_port) file.txt ccc@localhost:/tmp/

# 查詢 SSH 埠號（API）
curl http://localhost:8000/api/ssh/ccc
```

測試結果：
- ✅ SSH 登入成功（密碼 cccpass）
- ✅ SCP 檔案上傳成功
- ✅ `/etc/shadow` 密碼正確設定（yescrypt hash）
- ✅ 單元測試 8/8 通過
