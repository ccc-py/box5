# v2.0 帳號安全與管理系統（已完成）

## 目標

建立完整的帳號管理機制，包含信箱驗證、密碼重置、API Key 管理。

## 實作功能

### 1. 信箱驗證 ✅
- 註冊時輸入信箱，發送驗證連結
- JWT token（含過期時間 24 小時）
- 未驗證信箱限制：只能上傳、不能分享
- 可重發驗證郵件（每小時上限 3 次）

### 2. 密碼重置 ✅
- 「忘記密碼」頁面 → 寄送重置信連結
- Token 1 小時有效
- 重置後同步更新 Docker 容器密碼（`docker exec chpasswd`）

### 3. API Key 管理 ✅
- 使用者建立/刪除 API Key
- Key 格式：`box5_sk_` + 48 字元隨機（顯示前綴，儲存 hash）
- 權限：read / write / admin
- 可設定過期時間或手動撤銷

### 4. 登入記錄 ✅
- 記錄每次登入時間、IP、瀏覽器
- 顯示「最近的登入活動」

## API 端點

| Method | Endpoint | 說明 |
|--------|----------|------|
| POST | `/api/auth/register` | 註冊（寄送驗證信） |
| POST | `/api/auth/login` | 登入 |
| GET | `/api/auth/me` | 取得個人資料 |
| GET | `/api/auth/login-history` | 登入記錄 |
| POST | `/api/keys` | 建立 API Key |
| GET | `/api/keys` | 列出 API Keys |
| DELETE | `/api/keys/{id}` | 撤銷 API Key |
| GET | `/verify-email?token=xxx` | 驗證信箱頁面 |
| GET | `/forgot-password` | 忘記密碼頁面 |
| POST | `/forgot-password` | 請求密碼重置 |
| GET | `/reset-password?token=xxx` | 重置密碼頁面 |
| POST | `/reset-password` | 執行密碼重置 |
