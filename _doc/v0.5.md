# v0.5 規劃

## 目標
建立網頁版程式碼編輯器，類似 Web 版 VSCode，支援：
- 檔案/資料夾編輯
- 整合終端機（Terminal）
- WebSocket 即時通訊

## WebSocket 訊息格式

### 請求格式（XML）
```xml
<message type="terminal_input">
  <command>ls -la</command>
  <cwd>/project</cwd>
</message>
```

### 响应格式（XML + JSON）
```xml
<message type="terminal_output">
  <code>0</code>
  <output>{"stdout": "...", "stderr": "..."}</output>
</message>
```

```xml
<message type="file_content">
  <path>/project/file.py</path>
  <content>{"content": "...", "language": "python"}</content>
</message>
```

## 技術選型
- 前端編輯器：Monaco Editor
- 終端機：xterm.js + xterm-addon-fit
- 後端通訊：FastAPI WebSocket + HTTP REST API
- Shell 執行：Python subprocess
