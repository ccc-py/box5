現況分析
- 容器已有 zsh、vim、code5，但網路隔離無法外部 SSH
兩種方案
1. 容器內 SSH：每個容器跑 openssh-server，動態埠暴露到主機
2. 主機代理：主機 SSH → docker exec 進容器（更簡單）
推薦實作步驟
- Dockerfile 加入 openssh-server
- start.sh 同時啟動 SSH + FastAPI
- main.py 建立容器時用 useradd + chpasswd 建立系統使用者
- 動態分配 SSH 埠號，寫入 .ssh_port 檔案
- 新增 API /api/ssh/{username} 返回連線資訊
