# box5 v0.9

## New Features & Fixes
1. **PyPI Distribution config**: Mapped setuptools fields inside `pyproject.toml` and drafted `MANIFEST.in` to flawlessly bundle frontend static files, templates, `README.md`, and logic files into the public PyPI release distributions. 
2. **Terminal Raw PTY & Multiple Sessions**:
   - Shifted away from manual string parsing for shell inputs to piping raw byte strings to the pseudo-terminal. This immediately fixed duplicate echo problems and lost prompts. 
   - Overhauled `editor_ws.py` to route keyboard signals independently.
   - Introduced dynamic PTY multiplexing (`session_key`). This creates parallel terminal instances tied to unique folder subdirectories rather than squashing everyone per connection, unlocking precise "New Terminal" tab behavior.
3. **IDE Notification UI Cleanup**: Moved backend saving responses and errors out of the terminal display scope. It gracefully pops up standard UI tooltips now.
4. **Testing Enhancements**: Handshake and real-time execution flows for Websocket tests (`test_terminal_websocket`) deployed smoothly.
