# v2.4 進階安全與 MFA

## 目標

強化帳號安全，支援多因素驗證、Session 管理、IP 白名單。

## 功能

### 1. 多因素驗證（MFA）
- TOTP（相容 Google Authenticator）
- 掃描 QR Code 設定
- 啟用後登入需輸入 6 位數驗證碼
- 提供 10 組備援碼（下載保存，用過即刪）

### 2. Session 管理
- 同時多個 session（登入裝置列表）
- 檢視所有登入中的裝置
- 遠端登出（一鍵踢出所有其他 session）
- Session 過期時間：30 天

### 3. IP 白名單（可選）
- 設定允許存取的 IP 範圍
- 不在白名單的 IP 需額外驗證

### 4. 異常登入偵測
- 記錄登入 IP、位置、裝置
- 新地點/新裝置登入時寄送通知
- 短時間多次失敗自動鎖定（30 分鐘）

## 資料庫變更

```sql
ALTER TABLE users ADD COLUMN mfa_secret TEXT;
ALTER TABLE users ADD COLUMN mfa_enabled INTEGER DEFAULT 0;

CREATE TABLE mfa_recovery_codes (
    id INTEGER PRIMARY KEY,
    user_id INTEGER REFERENCES users(id),
    code_hash TEXT,
    used INTEGER DEFAULT 0
);

CREATE TABLE sessions (
    id INTEGER PRIMARY KEY,
    user_id INTEGER REFERENCES users(id),
    token_hash TEXT,
    ip TEXT,
    user_agent TEXT,
    device_name TEXT,
    created_at TEXT,
    expires_at TEXT,
    revoked INTEGER DEFAULT 0
);
```

## API 端點

| Method | Endpoint | 說明 |
|--------|----------|------|
| POST | `/api/mfa/setup` | 啟動 MFA 設定（取得 QR Code） |
| POST | `/api/mfa/verify` | 驗證 MFA 碼完成設定 |
| POST | `/api/mfa/disable` | 停用 MFA |
| GET | `/api/mfa/recovery-codes` | 取得備援碼 |
| POST | `/api/sessions` | 列出所有 session |
| DELETE | `/api/sessions/{id}` | 撤銷單一 session |
| DELETE | `/api/sessions` | 撤銷所有其他 session |
| GET | `/api/security/log` | 安全性日誌 |

## 前端頁面

- `/settings/security` — 安全設定
- `/settings/mfa` — MFA 設定精靈
- `/settings/sessions` — 登入裝置管理
