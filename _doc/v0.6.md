# v0.6 規劃

## 目標
修正 Editor Terminal 功能

## 問題描述
- WebSocket 連線返回 403 Forbidden 錯誤
- Terminal 輸入沒有回應

## 問題根源
1. Server/main.py 中 `app.include_router(router, prefix="/api")` 與 WebSocket 端點衝突
2. Server/main.py 中重複調用 `websocket.accept()`

## 修復方案

### 1. 移除重複的 accept 調用
- Server/main.py: 先調用 `websocket.accept()` 再調用 `websocket_endpoint()`
- Server/editor_ws.py: 移除 `websocket_endpoint()` 中的 `websocket.accept()`

### 2. Terminal 實現
- 改用簡化的命令執行模式：輸入命令後按 Enter 執行
- 使用 `subprocess.run()` 執行 shell 命令
- 前端實現命令緩衝區和提示符顯示

### 3. HTTP API 備用
- `/api/editor/files` - 列出目錄
- `/api/editor/file` - 讀取檔案
- POST `/api/editor/file` - 寫入檔案
