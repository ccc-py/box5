# v0.7 規劃

## 目標
為 Editor 加入功能表（Menu），支援檔案操作

## 功能需求

### 1. 功能表設計
- File 功能表：New, Open, Save, Save As, Close, Exit
- Edit 功能表：Undo, Redo, Cut, Copy, Paste
- View 功能表：Toggle Sidebar, Toggle Terminal
- Help 功能表：About, Keyboard Shortcuts

### 2. 檔案操作
- **New** - 建立新檔案（需輸入檔名）
- **Open** - 開啟檔案（從檔案總管選擇）
- **Save** - 儲存當前檔案（若無名稱則彈出儲存對話框）
- **Save As** - 另存新檔（需輸入檔名）
- **Close** - 關閉當前分頁（若有修改提示儲存）

### 3. 快捷鍵
- Ctrl/Cmd + S - 儲存
- Ctrl/Cmd + O - 開啟檔案
- Ctrl/Cmd + W - 關閉分頁
- Ctrl/Cmd + N - 新建檔案
- Ctrl/Cmd + Q - 離開

### 4. UI 設計
- 頂部菜單欄（Menu Bar）
- 下拉選單（Dropdown Menu）
- 鍵盤快捷鍵提示
- 未儲存檔案的 (*) 標記在分頁上
