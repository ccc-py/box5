# v2.2 分享功能

## 目標

讓使用者能分享檔案給非註冊用戶，並支援密碼保護與期限控制。

## 功能

### 1. 分享連結
- 對任意檔案產生分享連結
- Token 格式：`share_` + 32 字元隨機
- 連結可複製，分享統計計數

### 2. 分享設定
- 密碼保護（可選）
- 過期時間（可選）：1 小時 / 1 天 / 7 天 / 30 天 / 永不
- 下載次數限制（可選）

### 3. 分享頁面
- 公開分享：`/share/{token}` 直接下載
- 密碼保護：輸入密碼後下載
- 過期/超限：顯示錯誤訊息

### 4. 分享管理
- 我的分享列表
- 檢視次數、下載次數
- 撤銷分享連結

## 資料庫變更

```sql
CREATE TABLE shares (
    id INTEGER PRIMARY KEY,
    user_id INTEGER REFERENCES users(id),
    file_path TEXT,
    token TEXT UNIQUE,
    password_hash TEXT,
    expires_at TEXT,
    max_downloads INTEGER,
    download_count INTEGER DEFAULT 0,
    view_count INTEGER DEFAULT 0,
    created_at TEXT,
    revoked INTEGER DEFAULT 0
);
```

## API 端點

| Method | Endpoint | 說明 |
|--------|----------|------|
| POST | `/api/shares` | 建立分享連結 |
| GET | `/api/shares` | 我的分享列表 |
| GET | `/api/shares/{token}` | 取得分享資訊 |
| DELETE | `/api/shares/{id}` | 撤銷分享 |
| GET | `/api/shares/{token}/download` | 下載檔案（公開） |
| POST | `/api/shares/{token}/unlock` | 解鎖密碼保護 |

## 分享頁面設計

```
/share/{token}
├── 輸入密碼（可選）
├── 下載按鈕
├── 檔案資訊（名稱、大小）
└── 過期時間顯示
```
