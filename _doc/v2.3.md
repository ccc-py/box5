# v2.3 雙模式同步系統（已完成）

## 目標

支援兩種運行模式：
1. **Docker 模式**（現有）— 每個使用者一個容器，隔離環境
2. **本機模式**（新）— 直接連接遠端 Server，不需 Docker

讓本機指定資料夾同步遠端指定資料夾，無論是否使用 Docker。

## 架構

### 兩種模式

```
┌─────────────────────────────────────────┐
│           統一入口 (k8s/main.py)         │
│         自動偵測模式，統一 API             │
└──────────────────┬────────────────────────┘
                  │
        ┌─────────┴──────────┐
        ▼                     ▼
┌──────────────┐    ┌──────────────────┐
│  Docker 模式  │    │   本機模式        │
│              │    │                  │
│ 每用戶一個容器│    │ 直接 sync_folder  │
│ 隔離環境     │    │ 連接遠端 Server   │
│  port:xxxxx  │    │  可指定路徑       │
└──────────────┘    └──────────────────┘
```

### 模式偵測

```bash
BOX5_MODE=docker    # 預設，啟用 Docker 容器模式
BOX5_MODE=local     # 本機模式，直接 sync
```

## 本機模式功能

### 同步資料夾

```bash
# 啟動本機同步用戶端
box5-client --server http://box5.com --user ccc --folder ./my-sync --password xxxx

# 或設定環境變數
export BOX5_SERVER=http://box5.com
export BOX5_FOLDER=/path/to/sync
```

### 同步行為

- 使用 `watchdog` 監控本機資料夾變化
- 啟動時完整同步（pull）
- 變更時增量同步（delta）
- 支援上傳、下載、刪除
- 衝突處理：保留雙方版本（filename.ext vs filename.ext.1）

### 連線模式

```
本機資料夾 ←→ k8s Server (3111/8000)
              ↓
          sql5 資料庫
```

### API 呼叫

| 操作 | API |
|------|-----|
| 同步檔案列表 | `GET /api/files?folder=xxx` |
| 上傳 | `POST /api/files/upload` |
| 下載 | `GET /api/files/{id}/download` |
| 刪除 | `DELETE /api/files/{id}` |
| 同步歷史 | `GET /api/files/history/{filename}` |

## Docker 模式維持現有功能

- 每個使用者一個 Docker 容器
- 容器內跑獨立 Server
- 隔離環境，SSH 支援
- 本機 folder mount 到容器內 `/tmp/box5/sync`

## 實作

### 1. Client 模組 (`k8s/client.py`)

```python
class SyncClient:
    def __init__(self, server_url, username, password, folder)
    def start():            # 啟動 watchdog 監控
    def stop():             # 停止監控
    def sync():             # 全量同步
    def handle_change()     # 處理變化事件
    def upload_file():      # 上傳
    def download_file():    # 下載
```

### 2. Server API 改動

- 新增 `folder` 參數支援多資料夾同步
- 支援 `X-Sync-Hash` 差量同步（可選）

### 3. 模式切換

```python
# main.py
MODE = os.getenv("BOX5_MODE", "docker")

if MODE == "docker":
    # 現有 Docker 模式
    use_docker_mode()
else:
    # 本機模式
    use_local_mode()
```

### 4. CLI 入口

```bash
# Docker 模式（預設）
box5-sync --user ccc --folder ./sync

# 本機模式
box5-sync --local --server http://box5.com --user ccc --folder ./sync
```

## 環境變數

| 變數 | 說明 | 預設 |
|------|------|------|
| `BOX5_MODE` | 模式：docker / local | docker |
| `BOX5_SERVER` | Server URL（本機模式） | localhost:3111 |
| `BOX5_FOLDER` | 同步資料夾 | ./sync |
| `BOX5_USER` | 使用者名稱 | - |
| `BOX5_PASS` | 密碼 | - |
