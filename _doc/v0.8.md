# v0.8 規劃

## 目標
Editor 右鍵選單、儲存提示、相對路徑處理

## 功能需求

### 1. 右鍵選單 (Context Menu)
- 在 File Explorer 的檔案或資料夾上按右鍵，出現選單
- **Save** - 儲存檔案
- **Rename** - 重新命名（實際 rename，而非複製）
- **Delete** - 刪除檔案/資料夾

### 2. 關閉檔案提示
- 按 × 關閉分頁時，若有未儲存修改則提示儲存
- Close / Close All 功能也有儲存提示
- 提示對話框改用自訂對話框（非系統 alert/confirm）

### 3. 分頁切換修正
- 點擊不同分頁時，正確顯示該檔案內容
- 切換前先儲存當前編輯內容

### 4. View 功能修正
- Toggle Sidebar - 正確切換側邊欄顯示
- Toggle Terminal - 正確切換終端機顯示

### 5. 自訂對話框
- 所有 prompt/alert/confirm 改用自訂 HTML 對話框
- 支援：showAlert, showPrompt, showConfirm

### 6. Terminal 相對路徑
- 執行命令時，相對路徑自動轉為絕對路徑
- 解決 `./sync` 路徑不存在的問題

### 7. 相對路徑處理
- Editor API 支援相對路徑 `./sync`
- 伺服器端自動轉換為專案根目錄的相對路徑
