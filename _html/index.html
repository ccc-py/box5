元件說明

☁

Server

FastAPI 後端伺服器，提供使用者認證與檔案 CRUD API。

  • JWT 無狀態認證
  • sql5 資料庫（WebSocket 傳輸）
  • 檔案版本歷史
  • 公開檔案分享
  • WebSocket 編輯器後端
cd Server && uv run uvicorn main:app --port 3111
🖥

Client

桌面同步客戶端，監控本機資料夾自動同步至雲端。

  • watchdog 即時監控
  • MD5 雜湊去重
  • 支援 public/ 公開資料夾
  • 初始完整同步
uv run python -m Client.main --username USER --password PASS
🌐

Website

網頁介面，支援檔案瀏覽、檢視與線上編輯。

  • Jinja2 伺服器端渲染
  • Markdown / 圖片 / HTML 檢視
  • Monaco Editor 程式碼編輯器
  • xterm.js 終端機
  • 檔案版本歷史瀏覽
cd Website && uv run uvicorn main:app --port 3112
🐳

k8s（多租戶）

多租戶版本，每個使用者獨立的 Docker 容器。

  • Docker 容器隔離
  • 自動容器建立
  • Volume Mount 同步
  • SQLite WAL 模式
  • 內建 vim + code5（AI 編碼助手）
cd k8s && ./k8s_run.sh

快速開始

1

安裝相依套件

./install.sh

使用 uv 自動安裝所有 Python 相依套件與 Playwright 瀏覽器。

2

執行全部服務

./run.sh

同時啟動 Server（port 3111）、Website（port 3112）與 Client。

3

執行測試

./test.sh

執行單元測試、API 測試與 E2E 測試。

4

啟動 k8s 多租戶版本

cd k8s && ./k8s_run.sh

啟動多租戶 Docker 容器版本（port 8000）。預設帳號：ccc / cccpass
